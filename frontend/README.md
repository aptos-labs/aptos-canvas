# Aptos Canvas: Frontend

The frontend lets you do the following:
1. Create canvases.
1. View canvases.
1. Draw on canvases.
1. Perform various admin operations.

For token-related operations use a marketplace.

## Development
To get started, run:
```
pnpm install
pnpm start
```

To format your code:
```
pnpm fmt
```

## Deployment
The site is deployed automatically via GitHub Pages.
